# CodeAlpha - Language Translation Tool

A web-based Language Translation Tool built as part of the CodeAlpha Artificial Intelligence Internship. This tool provides a clean user interface where users can enter text, automatically detect the source language, and select a target language to receive real-time, accurate translations.

## Features
- **Auto-Detect Source Language:** Detects the language of the input text automatically.
- **Multiple Target Languages:** Supports translation into various languages like German, Spanish, French, Urdu, and more.
- **User Interface:** A responsive and easy-to-use web UI built with Flask and Bootstrap.
- **One-Click Copy:** Includes a copy-to-clipboard button for user convenience.

## Technologies Used
- Python
- Flask Framework
- Deep-Translator API (Google Translate)
- HTML5 / CSS3 / Bootstrap 5
